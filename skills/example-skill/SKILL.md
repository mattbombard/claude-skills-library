# Example Skill

This is an example skill that demonstrates the proper structure for creating Claude Code skills.

## Purpose

This skill serves as a reference implementation showing how to structure a skill with frontmatter, instructions, examples, and guidelines.

## Instructions

When this skill is invoked:

1. Acknowledge the skill has been activated
2. Explain what you would do if this were a real skill
3. Show the user the structure of this example
4. Suggest how they might create their own skills

## Examples

**Example 1: Basic Invocation**
```
User: /example-skill
Claude: The example skill has been activated. This demonstrates the basic structure...
```

**Example 2: With Arguments**
```
User: /example-skill show structure
Claude: Here's the structure of a Claude Code skill...
```

## Guidelines

- Keep skill names lowercase with hyphens
- Write clear, actionable descriptions
- Provide specific instructions, not vague guidance
- Include realistic examples
- Consider edge cases and error scenarios

## Notes

This example can be removed once you've created your own skills. It's here to help you understand the format.
